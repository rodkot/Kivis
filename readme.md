## Описание

Desktop приложение для рисования двумерных изображений с помощью линий, штампов и заливок.
## Стек

```Jetpack Compose, Kotlin```

## Фичи-лист

- Рисование линий
  - Алгоритм Брезенхэма
- Рисование правильных прямоугольников 
- Заливка 
  - Алгоритм Span-заливки
- Ластик 
- Открытие/Сохранение файлов
